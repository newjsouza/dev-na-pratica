Como automatizar tarefas chatas com Python em 15 minutos

1 min de leitura #python#automacao

Python é a linguagem mais amigável para automação. Vou te mostrar 3 scripts que você pode escrever agora e usar hoje mesmo.

Script 1: Renomear arquivos em massa

import os
pasta = "/caminho/para/suas/fotos"
for i, arquivo in enumerate(os.listdir(pasta)):
    if arquivo.endswith(".jpg"):
        novo_nome = f"foto_{i+1:03d}.jpg"
        os.rename(os.path.join(pasta, arquivo), os.path.join(pasta, novo_nome))

Isso renomeia todas as fotos .jpg para foto_001.jpg, foto_002.jpg etc.

Script 2: Backup automático

import shutil, datetime
origem = "/home/seu/projeto"
destino = f"/backups/projeto_{datetime.date.today()}"
shutil.copytree(origem, destino)
print(f"Backup criado em {destino}")

Script 3: Enviar e-mail com relatório

import smtplib
from email.message import EmailMessage

msg = EmailMessage()
msg.set_content("Relatório gerado automaticamente.")
msg["Subject"] = "Relatório Diário"
msg["From"] = "seu@email.com"
msg["To"] = "chefe@empresa.com"

with smtplib.SMTP("smtp.gmail.com", 587) as s:
    s.starttls()
    s.login("seu@email.com", "senha_app")
    s.send_message(msg)

Como começar

Crie um arquivo .py com o código, execute com python3 script.py. Em 15 minutos você tem sua primeira automação rodando.
